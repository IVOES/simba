DHT
=======

About
-----

Read and print the temperature and humidity measured with DHT sensor.

See the :doc:`DHT sensor Library Reference
<../library-reference/drivers/sensors/dht>` for more details.

Source code
-----------

.. include:: dht/source-code.rst

The source code can also be found on Github in the
:github-tree:`examples/dht` folder.

Build and run
-------------

Build and run the application.

.. code-block:: text

   $ cd examples/dht
   $ make -s BOARD=<board> run
   Temperature: 22.5 C
   Humidty:     68 %RH
